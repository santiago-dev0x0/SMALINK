# SMALINK - URL Shortener 

## 📖 Descripción
Plataforma  para acortar URLs con análisis de rendimiento.

## 🚀 Quick Start
```bash
# Backend
cd server && npm install && npm run start:dev

# Frontend  
cd client && npm install && npm run dev
```

## 📁 Estructura
- `/server` - Backend NestJS
- `/client` - Frontend React
